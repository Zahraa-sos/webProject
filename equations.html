What Is An Equation?

An equation is a mathematical sentence that has two equal sides separated by an equal sign.(See the example below:)
x+1 = 4
That equations says:

what is on the left (x + 1) is equal to what is on the right (4)

Equation Parts:

  • Variable: A symbol for a number we don't know yet, often represented by letters like x or y.
  • Constant: A number on its own, without any variable, which does not change.
  • Coefficient: A number used to multiply a variable. For example, in the expression 4x, 4 is the coefficient.
  • Implicit Coefficient: When a variable is written without a number next to it, it's assumed to have a coefficient of 1. For instance, x is equivalent to 1x.

Types Of Equations

Based on the degree, equations can be classified into three types. Following are the three types of equations in math:

  • Linear Equations: an equation in which the highest power of the variable is always.

  • Quadratic Equations: polynomial equations of degree 2 in one variable of type f(x) = ax2 + bx + c = 0 where a, b, c, ∈ R and a ≠ 0.

  • Cubic Equations: is an algebraic equation of degree three and is of the form ax3 + bx2 + cx + d = 0.

How to Solve an Equation?

-move everything except "x" (or whatever name the variable has) over to the right hand side.

Example:
Solve 3x−6 = 9

  1. Start with: (3x - 6 = 9)
  2. Add 6 to both sides: (3x = 9 + 6)
  3. Divide by 3: (x = (9 + 6)/3)
  4. The answer is:x = 5

What is a Solution?

A Solution is a value we can put in place of a variable (such as x) that makes the equation true.

Example:

x − 2 = 4

When we put 6 in place of x we get:
6 − 2 = 4
which is true.
So x = 6 is a solution.

How about other values for x?

  • For x=5 we get "5−2=4" which is not true, so x=5 is not a solution.
  • For x=9 we get "9−2=4" which is not true, so x=9 is not a solution.

    • In this case x = 6 is the only solution.

More Than One Solution

There can be more than one solution.

Example: (x−3) (x−2) = 0

When x is 3 we get:
(3−3) (3−2) = 0 × 1 = 0
which is true

And when x is 2 we get:
(2−3) (2−2) = (−1) × 0 = 0
which is also true

So the solutions are:
x = 3, or x = 2

When we gather all solutions together it is called a Solution Set
The above solution set is: {2, 3}
